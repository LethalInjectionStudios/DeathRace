# DeathRace
Remake of Death Race arcade game

Currently the resolution is messed up and full screen mode is kinda screwy. Will be fixed at some point.

# Controls
White Car <br>
Forward: Numpad 8 <br>
Left: Numpad 4 <br>
Right: Numpad 6 <br>
Boost: Numpad 7 <br>
Fire: Right Shift <br>
<br>
Grey Car <br>
Forward: W <br>
Left: A <br>
Right: D <br>
Boost: E <br>
Fire: Space Bar<br>
